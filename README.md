# 📝 Task Manager App

A full-stack **Task Manager App** built with **Next.js (App Router)**, **Prisma ORM**, and **SQLite**.  
Features:
- ✅ Add tasks
- ✅ Edit tasks
- ✅ Delete tasks
- ✅ Task detail page
- ✅ Status & description management
